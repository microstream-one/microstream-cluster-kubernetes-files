# MicroStream Cluster Kubernetes Files

Files required for deploying an Eclipse-Store Cluster in an On-Premises Kubernetes Cluster.

More information about the contents of this repository and the different installation methods can be found in  [the docs.](https://docs.microstream.one/enterprise/manual/1/on-premises/on-prem-single-cluster.html)

## Branches

- _master_ - Contains the yaml files required for installing via `kubectl`
- _helm-chart_ - Contains the helm chart required for installing via `helm`
